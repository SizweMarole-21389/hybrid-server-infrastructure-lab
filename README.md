# Hybrid Server Infrastructure Deployment & Security Hardening

## Overview
This hands-on lab project covers the full deployment and security hardening of a hybrid server environment using Microsoft Hyper-V, Active Directory, and Windows Server Storage Spaces. The project documents real configuration steps with supporting screenshots across two major modules.

## Lab Modules

### Module 1 — File Server Resource Manager (FSRM) & Print Management Security
Configuration and hardening of file server resources, quota management, file screening, and print server security policies.

| Section | Contents |
|---|---|
| Part I | Initial FSRM setup and quota configuration |
| Part III | File screening policies and storage reports |
| Part V | Print management security and audit logging |

### Module 2 — Storage Spaces, Data Deduplication & Server Backup Management
Deployment of Windows Storage Spaces for resilient storage, data deduplication configuration, and server backup strategy implementation.

| Section | Contents |
|---|---|
| Part I | Storage Spaces pool creation and virtual disk provisioning |
| Part II (A & B) | Data deduplication setup and validation |
| Part III | Backup policy configuration |
| Part IV | Backup verification and restoration testing |

## Technologies & Tools
- **Microsoft Hyper-V** — virtualisation platform for lab environment
- **Active Directory Domain Services (AD DS)** — domain management and GPO enforcement
- **Windows Server Storage Spaces** — software-defined storage with resiliency
- **File Server Resource Manager (FSRM)** — quota, screening, and reporting
- **Windows Server Backup** — incremental backup and recovery
- **Print Management Console** — centralised print server security

## Skills Demonstrated
- Windows Server deployment and configuration
- Storage architecture and resilience planning
- Security policy implementation via Group Policy
- File server hardening and access control
- Backup and disaster recovery planning
